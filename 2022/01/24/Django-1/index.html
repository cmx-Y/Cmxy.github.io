URL

Uniform Resource Locator

定义：统一资源定位符

作用：表示互联网上某个资源的地址

统一格式：

protocol://hostname[:port]/path[?query][#fragment]

  • protocol - 协议
  • hostname - 主机名
  • path - 文件路径
  • ?query 参数
  • #fragment 锚点，定位文章的某个位置

eg：

https://www.bilibili.com/video/BV1vK4y1o7jH?p=4

通过p=4，会定位到视频P4

https://docs.djangoproject.com/en/4.0/#the-template-layer

通过the-template-layer，会定位到文档中相应位置
