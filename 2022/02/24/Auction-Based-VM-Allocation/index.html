Auction_Based VM Allocation

发表于 2022-02-24 分类于 paper_reading

Abstract

  • ECNs: Edge Cloud Nodes
    • the computation and storage services of remote cloud data centers are moved to Edge Cloud Nodes
    • so Edge cloud computing is possible
  • The problem:
    • allocating Virtual Machine(VM) resources in ECNs to mobile users by using the auction theory
    • ECNs – sellers mobile users – buyers

Introduction

  • mobile cloud computing

    • migrate various applications from smart devices to remote cloud data centers

  • edge cloud computing

    • a number of small-scale computing and storage servers are placed at network edges to form some Edge Cloud Nodes

​ In this paper, we focus on the VM resource allocation for deadline-sensitive cloud computing tasks in distributed edge cloud, attempting avoid overwork and wasted resource in any of ECNs.
